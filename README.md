# react-demo project

### getting started

🧶 make sure to have yarn installed
: `npm i -g yarn`

install dependencies
: `yarn`

### scripts

🎬 `yarn start`
: builds and starts up a dev [server](http://localhost:8000)

👷 `yarn build`
: builds in production mode

🚀 `yarn deploy`
: deploys to github pages!

🧼 `yarn clean`
: cleans /dist folder
