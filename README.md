# web-project-outline

### getting started

🧶 make sure to have yarn installed
: `npm i -g yarn`

install dependencies
: `yarn`

### scripts

🎬 `yarn start`
: builds and starts up a dev [server](http://localhost:8000)

👀 `yarn watch`
: builds in watch mode

🚀 `yarn build`
: builds in production mode

👷 `yarn build-dev`
: builds in development mode

🧼 `yarn clean`
: cleans /dist folder
